Remarks

            The format is a composite format string that uses the same format as
            Format  method. The method parameter, if
            not null, is printed before the message; it typically is used to identify the
            method in which a warning is detected.
            

            The clientData parameter can be anything. Its value and meaning is
            defined by an application and not the library.
